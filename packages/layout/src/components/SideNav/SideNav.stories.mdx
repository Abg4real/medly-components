# SideNav

### Basic SideNav

Sidenav layout componenet could be pretty handy to show menu items.

```tsx
<SideNav>
    <SideNav.NavItem path="/home">
        <SideNav.NavIcon>
            <HomeIcon />
        </SideNav.NavIcon>
        <SideNav.NavText>Home</SideNav.NavText>
    </SideNav.NavItem>
</SideNav>
```

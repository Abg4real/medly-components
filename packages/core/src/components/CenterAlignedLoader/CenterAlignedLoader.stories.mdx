# CenterAlignedLoader

This component takes the parent's full width and height and then render the loader in the center. Although it can take any react component as loader, but you can cehckout all the loaders in [Medly Components](https://medly.github.io/medly-components/?path=/story/loaders--loaders)

    </Story>
</Preview>

#### Example with different loader

You can pass any React component as loader like this `<CenterAlignedLoader withLoadingBox loader={<ConcentricCircleLoader size="XS" />} />`

### Passing component to a row

If you want to render any cell according to your design you can pass the component in column config like below

```ts
const columns: ColumnConfig[] = [
    { title: 'Patient Info', field: 'patientInfo', sort: true, component: StatusFormatter },
    { title: 'RxInfo', field: 'rxInfo', sort: true },
    { title: 'Price', field: 'price', sort: true, component: StatusFormatter },
    { title: 'Status', field: 'status', sort: true, component: StatusFormatter, align: center }
];
```

#### Column Config Prop Types

You can use Table Prop types like below to typesafe your column config.

```ts
import { GetComponentProps } from '@medly-components/utils';
import { CardTable } from '@medly-components/core';

type CardTableProps = GetComponentProps<typeof CardTable>;

type ColumnConfig = CardTableProps['columns'];
```

### With column hide feature

To use the column hide feature use `Table.SelectableColumns` component and save the column config in state variable and pass the same column config to `Table` and `Table.SelectableColumns` component.

```tsx
export const WithColumnHide = () => {
    const [columnConfig, setColumnConfig] = useState(columns);

    return (
        <>
            <Table.SelectableColumns columns={columnConfig} fieldToChange="hide" onChange={setColumnConfig} />
            <Table data={data} columns={columnConfig} />
        </>
    );
};
```

### Table Prop Types

You can use Table Prop types like below to typesafe your column config.

```ts
import { GetComponentProps } from '@medly-components/utils';
import { Table } from '@medly-components/core';

type TableProps = GetComponentProps<typeof Table>;

type ColumnConfig = TableProps['columns'];
```

# Drawer

Drawer component. You can play with the component in the canvas tab.

Drawer consists of three sections Header, Content and Footer. All of these you can directly use from Drawer component only like below ...

```typescript
<Drawer open={drawerState} onClose={changeDrawerState}>
    <Drawer.Header>Add Filters</Drawer.Header>
    <Drawer.Content>
        CONTENT
    </Drawer.Content>
    <Drawer.Footer>
        <Button variant="flat" onClick={onClearAll}>
            Clear All
        </Button>
    </Drawer.Footer>
</Drawer>
```

Shadow will appear on the Header & Actions section if you scroll the content.

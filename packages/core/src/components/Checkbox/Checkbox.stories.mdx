# Checkbox

Simple checkbox component which allows users to select options from a set. It's a controlled react component. So you have to have the handle state like below.

```tsx
const [isChecked, setChecked] = useState(false),
    onChangeHandler = (e: React.ChangeEvent<HTMLInputElement>) => setChecked(e.target.checked);

return <Checkbox label="Lorem Ipsum" checked={isChecked} onChange={onChangeHandler} />;
```

# Popover

You have to use both `Popover` & `PopoverWrapper` component.

```typescript
import { PopoverWithBackground, PopoverWrapper } from '@medly-components/core';

export const WithClickInteraction = () => {
    const [isPopoverVisible, setPopoverVisibilityState] = useState(false);

    const showPopover = () => setPopoverVisibilityState(true),
        hidePopover = () => setPopoverVisibilityState(false);

    return (
        <DummyWrapper>
            <PopoverWrapper
                showPopover={isPopoverVisible}
                onClick={showPopover}
                onOuterClick={hidePopover}
                placement={select('Placement', placement, 'top')}
            >
                <DummyDiv>Hello</DummyDiv>
                <PopoverWithBackground fullWidth={boolean('Popover Full Width', false)} fullHeight={boolean('Popover Full Height', false)}>
                    <DummyPopoverContent>This is Popover</DummyPopoverContent>
                </PopoverWithBackground>
            </PopoverWrapper>
        </DummyWrapper>
    );
};
```

### Popover with click interaction

We have to use `onClick` and `onOuterclick` functions to set popover state.

### CardsTable Prop Types

You can use Table Prop types like below to typesafe your column config.

```ts

import { CardsTable } from '@medly-components/core';

type CardsTableProps = GetComponentProps<typeof CardsTable>;

type ColumnConfig = CardsTableProps['columns'];
```

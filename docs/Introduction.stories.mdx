## Getting Started

```sh
npx create-react-app medly-components-demo --typescript
cd medly-components-demo
npm i styled-components
npm i https://medly-artifacts.s3.amazonaws.com/medly-components/medly-components-theme-1.2.0.tgz
npm i https://medly-artifacts.s3.amazonaws.com/medly-components/medly-components-utils-1.2.0.tgz
npm i https://medly-artifacts.s3.amazonaws.com/medly-components/medly-components-icons-1.2.0.tgz
npm i https://medly-artifacts.s3.amazonaws.com/medly-components/medly-components-core-1.2.0.tgz
npm i https://medly-artifacts.s3.amazonaws.com/medly-components/medly-components-layout-1.2.0.tgz
npm i https://medly-artifacts.s3.amazonaws.com/medly-components/medly-components-loaders-1.2.0.tgz
```

Replace content of `index.tsx` with the following:

```tsx
import { store } from '@store';
import { defaultTheme } from '@theme';
import { ThemeProvider } from '@utils';
import React from 'react';
import * as ReactDOM from 'react-dom';
import { Provider } from 'react-redux';
import App from './App';

ReactDOM.render(
    <Provider store={store}>
        <ThemeProvider theme={defaultTheme}>
            <App />
        </ThemeProvider>
    </Provider>,
    document.getElementById('root')
);
```

Replace content in `App.tsx` with the following:

```tsx
const App: React.FC = () => (
    <div>
        <Button variant="flat" color="primary">
            Click me!
        </Button>
    </div>
);
```

## Packages

Library consists of five packages:

### [Core](https://github.com/medlypharmacy/medly-components/tree/master/packages/core)

Core package consits of components almost all basic react components like Avatar, Button, Input, List, Modal etc.

**Example**:

<List variant='horizontal'>
<Button variant="flat" color="secondary">
    Flat Button
</Button>

<Button variant="outlined" color="primary">
    Outlined Button
</Button>

<Button variant="solid">Solid Button</Button>
    </List>

### [Icons](https://github.com/medlypharmacy/medly-components/tree/master/packages/icons)

Icons package consists of most commonly used svg icons for add, delete, clearfile, clipboard etc exposed as react components.

**Example**:

<EditIcon size="M" color="pink" open />

### [Layout](https://github.com/medlypharmacy/medly-components/tree/master/packages/layout)

Layout package consists of components that help in creating a page layout using side panels and nav items, which are exposed as react components.

**Example**:

```tsx
<SideNav>
    <SideNav.NavList>
        <SideNav.NavItem>
            <SideNav.NavIcon>
                <HomeIcon />
            </SideNav.NavIcon>
            <SideNav.NavText>Home</SideNav.NavText>
        </SideNav.NavItem>
    </SideNav.NavList>
</SideNav>
```

### [Theme](https://github.com/medlypharmacy/medly-components/tree/master/packages/theme)

Themes lets you define how a component should be styled across website. This can be done individually for particular type of components, eg all icons should be solid or can be done for the whole website, like color theme for all type of components across website should be orange.

### [Utils](https://github.com/medlypharmacy/medly-components/tree/master/packages/utils)

Utils package consists of most commonly used functionalities or components. Eg [String Helper](https://github.com/medlypharmacy/medly-components/tree/master/packages/utils/src/stringHelpers.ts) has functions to convert a string to camelCase, case insensitive string comparison etc

## Storybook

This library has a story book which has following things in it for all components provided:

-   Preview of how component looks
-   Knobs and Props for the component
-   Docs that list default props and allowed values, default themes etc
